# create-initer

> An easy way to create various configuration files

一个用于生成各种配置文件的简易工具

## 使用

```sh
# pnpm
pnpm create initer@latest

# npm
npm create initer@latest

# yarn
yarn create initer@latest
```

## 已支持的配置文件

> 功能正在完善中，欢迎提供更多的配置文件
